한국전력 VR 업로드 API
===================

본 사이트는 한국전력 VR 프로젝트에서 업로드 기능을 3rd Party에게 제공하기 위한 RESTFUL API 문서입니다.

.. warning:: 모든 API는 개발사의 진행에 따라 업데이트 될 수 있습니다.

RESTFUL API
-----------

모든 API의 응답은 json 형태로 전달됩니다. XML 포맷은 제공하지 않습니다.
